Aplicação web de gerenciamento de tarefas desenvolvida com JavaScript puro, HTML5 e CSS3. Além de adicionar e remover tarefas, o projeto conta com edição, pesquisa por texto, filtro por status e persistência de dados via localStorage — as tarefas continuam salvas mesmo após fechar o navegador.
<img width="1852" height="942" alt="Tela to-do-list" src="https://github.com/user-attachments/assets/3b5fc4cf-ad11-48e8-9489-e6e18be034b6" />


## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Editar texto de tarefas existentes
- Remover tarefas individualmente
- Pesquisar tarefas por texto em tempo real
- Filtrar por status: todas / feitas / a fazer
- Persistência de dados via localStorage
- Interface responsiva

## Tecnologias utilizadas

- **HTML5** — estrutura e formulários
- **CSS3** — estilização, transições e responsividade
- **JavaScript** — manipulação do DOM, eventos e localStorage
- **Font Awesome** — ícones dos botões de ação

## Como executar localmente

```bash
# Clone o repositório
git clone https://github.com/annielemarinho/to-do-list.git

# Acesse a pasta
cd to-do-list
```

Abra o arquivo `index.html` diretamente no navegador. Nenhuma dependência ou instalação é necessária.

## Aprendizados

Este projeto foi desenvolvido durante os estudos de JavaScript, com foco em:

- Manipulação do DOM com `querySelector`, `createElement` e eventos
- Persistência de dados com `localStorage` (salvar, carregar, atualizar e remover)
- Alternância de formulários e estados visuais com `classList.toggle`
- Busca e filtragem dinâmica de elementos na página
- Organização do código em funções reutilizáveis
